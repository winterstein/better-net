# The problem:
Fake news, 
manipulative fake profiles,
misinformation campaigns,
toxic media (anger and name-calling),
are undermining trust in news and harming democratic processes worldwide.

Broken right now:
The above are all real.
E.g. Russia is known to run fake social media to interfere with western democracy.
Social media debates are increasingly polarised and toxic

Catastrophe: 
If elections are effectively controlled by bot networks.
If toxic debate drowns out reasoned debate - blocking vital coordinated action.

Why not fixed already?
Fixing this is hard.
Needs mainstream non-partisan action.
The status quo e.g. Facebook are largely ambivalent and don't wish to spend money on this.

Who loses if this is solved?
Russia, Elon Musk, Trump, influencers who thrive on anger

Who else is working on related problems?
Fact-check orgs and journalists.
What gap aren't they filling?
Ease of use. Value for the user.
E.g. sites where the user has to pro-actively go and check facts are not going to cut through.
Scale. Automation
Tackling bias and toxicity (some do, but the main focus is fact-checking).
Tackling social media (some do, but the main focus is news sites).

# The solution

BetterNet
Like ad-blocker for fake news, scams, doom-scrolls, and toxic manipulative content.

AI is the new magic ingredient that can make this work.
Value to the user is key.

Might bundle an ad-blocker and privacy tools for convenience to the user. But not in the MVP. You can get those elsewhere.

A browser plugin and mobile browser to improve quality of life online.
Similar to ad-blockers and privacy-browsers like duck-duck-go.
Built in AI that:
 - checks for fake news, scams, AI slop, bias, poor-reasoning, toxicity, doom-scrolling
 - shows nutrition labels as gentle nudges to the user (the user must remain in control)

Use a local small-LLM for fast private analysis
+ anonymous-use database for efficiency and community moderation
(how to prevent poisoning the shared database? not needed for MVP
maybe: clients query urls + headline + post content being analysed.
the most requested articles and topics get high-quality AI central processing
)

twitter community notes algorithm: looks for left+right agreement (robust against vote-packing)

MVP: browser plugin for chrome/edge/mozilla
local model classifier
option to use OpenAI/Anthropic API instead of local
anonymous performance stats

beachhead users: tech-savvy
then: like ad-blockers, this can reach a wide digitally literate group.

What's been tried before?
e.g. Full-Fact working via Twitter / Facebook -- when they felt they needed to do fact-checking.
